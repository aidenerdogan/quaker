# Notices

Quaker CLI is an open-source macOS maintenance engine by Aiden Ahmet Erdogan.

Quaker adds a local memory engine, user-defined policy rules, reusable profiles, event hooks, suggest-only scheduled recommendations, Quaker-branded CLI entrypoints, and product architecture for future GUI/TUI applications.
